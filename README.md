# python-lab# Лабораторная работа №1. Знакомство

## Задание
Скачать архив с 11 заданиями, выполнить их, оформить отчёт, сделать commit и push.

## Описание проделанной работы
- Создан репозиторий на GitHub.
- Репозиторий склонирован на ПК.
- Выполнены все 11 заданий по Python (строки, списки, словари, множества, срезы).
- Решения сохранены в соответствующих файлах `00_distance.py` ... `10_store.py`.
- Изменения закоммичены и отправлены на GitHub.

## Шпаргалка по работе с командами git
| Команда | Описание |
|---------|----------|
| `git clone <url>` | Склонировать репозиторий |
| `git add .` | Добавить все файлы в индекс |
| `git add <файл>` | Добавить конкретный файл |
| `git commit -m "..."` | Создать коммит с сообщением |
| `git push` | Отправить изменения на GitHub |
| `git status` | Проверить состояние репозитория |
| `git config --global user.name` | Установить имя пользователя |
| `git config --global user.email` | Установить email |
